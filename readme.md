# starAlign

> multiple sequence alignment for RNA/DNA sequences

## How to use?

1. git clone this repository
   ```bash
    git clone git@github.com:Guuhua/centerstar_MSA.git
   ```

2. make
   ```bash
    cd centerstar_MSA
    make
   ```

3. run
   ```bash
    ./staralign ./test/rna.fasta ./test/rna.align
   ```

clean
```bash
make clean
```
